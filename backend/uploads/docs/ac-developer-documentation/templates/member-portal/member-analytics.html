Member Analytics


Overview

This Application pulls Data from the "Member_Data_Consolidated" module using the search records API and uses Chart.js to visualize the data. Users can filter the data by the type of data (i.e. DWP, PIF,  Incurred Losses), Agency Report Carrier (i.e. Progressive, Nationwide, Guard...),  Agency Report Year, Agency Report Month, and or the Agency(ies) related to the Agency Report. The Application also includes a "Most Recent Report Months" Tab where the user can view the most recent report month for each carrier in their system. For more information on how the Agency Reports (Member_Data_Consolidated) is marked most recent, please reference the File Parser documentation.


Dependencies

  • JQuery
  • Lodash
  • Chart JS
  • Zoho Embedded App SDK


How It Works

This is a Client Side only Application using the Zoho Embedded SDK. 

  • Pull all required data using Zoho Get functions
  • Insert default data into Chart JS Objects
  • If the User filters data, the select change functions get all the values in the selectors recursively and filters the data accordingly by passing data through the "generalFunction" and then the "abstractChangeChart" function within the "generalFunction".
  • The "abstractChangeChart" Function includes error handling and some small user interface controls to hide charts with no data.
