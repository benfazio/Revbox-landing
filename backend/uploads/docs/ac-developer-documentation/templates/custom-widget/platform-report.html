Platform Report


Overview

This Application pulls all data from the "Platforms" module using "getAllRecords" and then filters the data to just records that have the same name as the name of the current user. Data from the "Platform_Percentage" module is the pulled using a "searchRecord" function that gets records where the Platform lookup field in the "Platform_Percentage" module matches the Platform IDs we got from the Platforms Module. Using the data we pulled from the "Platform_Percentage" module we can grab the corresponding Account/Agency ID from each record and then use that ID in a "getRecord" function to the "Accounts" module where we will grab our Accounts/Agency Data. The Application then displays the required data in a table format for the user to view.


Dependencies

  • JQuery
  • Lodash
  • Zoho Embedded App SDK
